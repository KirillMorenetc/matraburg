Матрас Passion (страсть) создан специально для энергичных людей любой весовой категории. Парные пружины с дополнительными ребрами жесткости способны нести повышенную нагрузку на один квадратный метр. Под практичным стеганым чехлом располагается латексированный конский волос, добавляющий прочности мягким слоям матраса. Настил из латекса позволяет поверхности идеально повторять все изгибы Вашего тела, поддерживая поясничную зону, расслабляя грудной отдел позвоночника. Изоляционным материалом служит полотно из натуральных волокон (войлок). По периметру изделие усилено коробом ППУ, что продлевает срок его службы и увеличивает зону комфортности.


Это матрас не только для отдыха, но и для любви!

Позвольте Вашей страсти бушевать ураганом, всё подчиняя своей воле...


Высокий уровень гигиеничности поддерживается за счет применения съемных чехлов, которые можно чистить и стирать.



Модель матраса выполняется в двух видах чехлов:

Чехол Classic Double Zip выполнен из прочного смесового двойного жаккарда с молнией.

Интересный дизайнерский ход – разделение чехла молниями на три части, позволит чистить полотнища отдельно по мере необходимости.


Чехол Luxe выполнен из прочного смесового стрейча (трикотажное полотно) высокой плотности с вискозой.

Молния-застежка по периметру позволяет легко снимать чехол (по частям) и при необходимости стирать.

Состав

  • Несъемный чехол из хлопкового стрейча, h=15 мм

  • Настил из латексированного кокосового волокна, h=10 мм

  • Комфортный настил из матропены, h=20 мм

  • Блок независимых пружин Multy Pocket, h=140 мм, 1000 пружин на спальное место

  • Короб из матропены по периметру матраса

  • Аксессуары: ручки, вентиляционные клапаны, фирменная этикетка
